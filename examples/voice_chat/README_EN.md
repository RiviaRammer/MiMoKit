## Features

- Speech Recognition (ASR): Real-time voice-to-text
- Text Chat (Chat): Multi-turn conversation with large language model
- Text-to-Speech (TTS): Convert replies to speech playback
- Voice Activity Detection (VAD): Automatic detection of speech start and end
- Streaming Playback: TTS supports streaming playback for reduced latency
